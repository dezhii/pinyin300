# 拼音小游戏

一个面向小学生的拼音闯关网页游戏：300 个常见汉字，100 关，每关 3 个汉字。孩子可以选择小火车、打怪兽、拼音花园、拼音星球等不同游戏风格；看到汉字后先听 3 遍读音，再输入不带声调的拼音。答错的汉字会自动加入错题本，方便复习。

## 已实现功能

- 4 种可选择游戏风格：拼音小火车、拼音打怪兽、拼音花园、拼音星球
- 300 个常见汉字拼音数据
- 100 关关卡地图，每关 3 个汉字
- 当前汉字自动朗读 3 次
- 支持本地标准 mp3 音频优先播放，缺失时自动退回浏览器朗读
- 输入拼音，不要求声调
- 支持 `ü` 的友好输入：如 `lǜ` 可输入 `lv` 或 `lu`
- 答错自动加入错题本
- 错题本查看、删除、清空、标记掌握
- 错题复习模式：每次复习 3 个错题，连续答对 2 次后自动标记已掌握
- 本地保存学习进度、关卡星级、错题记录和已选择游戏风格
- 响应式界面，适配电脑、平板和手机浏览器
- A4 打印版 300 字拼音表，拼音在汉字上方，点击字卡可朗读 3 次

## 技术栈

- React 18

- HTMLAudioElement 本地音频播放

- 原生 CSS

## 朗读说明

当前版本已经支持本地标准音频优先播放。把 300 个 mp3 文件放到 `public/audio/hanzi/`，并确保文件名与 `public/audio/hanzi/manifest.csv` 中的 `fileName` 一致即可。

如果对应音频不存在或播放失败，游戏和 A4 点读页都会自动退回浏览器 Web Speech API 朗读。不同浏览器、操作系统的中文语音可能不同，所以正式使用时建议补齐本地音频。

## 数据说明

当前 300 字来自小学低年级常见字范围，拼音数据保存在 `src/data/characters.ts`。多音字会展示多个拼音并接受多个答案；默认输入模式不要求声调。

音频清单生成命令：`npm run audio:manifest`。

音频缺失检查命令：`npm run audio:check`。在 300 个 mp3 补齐之前，这个命令会列出缺失文件并以失败状态退出，这是预期行为。

## A4 打印版

生成 A4 拼音表：`npm run print:a4`。

生成文件：

- `public/print/pinyin300-a4.html`
- `public/print/pinyin300.csv`

开发服务器启动后，可以访问 `/print/pinyin300-a4.html`。页面中拼音显示在汉字上方，点击任意字卡会朗读 3 次；点击页面上的「打印 A4」按钮可打印。
