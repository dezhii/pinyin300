# 拼音小火车

一个面向小学生的拼音闯关网页游戏：300 个常见汉字，100 关，每关 3 个汉字。孩子看到汉字后先听 3 遍读音，再输入不带声调的拼音。答错的汉字会自动加入错题本，方便复习。

## 已实现功能

- 拼音小火车主题首页
- 300 个常见汉字拼音数据
- 100 关关卡地图，每关 3 个汉字
- 当前汉字自动朗读 3 次
- 输入拼音，不要求声调
- 支持 `ü` 的友好输入：如 `lǜ` 可输入 `lv` 或 `lu`
- 答错自动加入错题本
- 错题本查看、删除、清空、标记掌握
- 错题复习模式：每次复习 3 个错题，连续答对 2 次后自动标记已掌握
- 本地保存学习进度、关卡星级、错题记录
- 响应式界面，适配电脑、平板和手机浏览器

## 技术栈

- React 18
- TypeScript
- Vite 5
- Web Speech API
- localStorage
- 原生 CSS

## 朗读说明

当前版本使用浏览器 Web Speech API 朗读汉字，开发和运行都很轻量。不同浏览器、操作系统的中文语音可能不同；如果后续要做到多音字和声调完全稳定，建议为每个汉字补充标准音频文件，并使用数据中的 `audio` 字段进行播放。

## 数据说明

当前 300 字来自小学低年级常见字范围，拼音数据保存在 `src/data/characters.ts`。多音字会展示多个拼音并接受多个答案；默认输入模式不要求声调。
