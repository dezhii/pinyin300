# 汉字标准音频目录

把 300 个汉字的本地标准发音 mp3 文件放在这个目录下。

文件名必须和 `manifest.csv` / `manifest.json` 中的 `fileName` 一致，例如：

- `001_yi1.mp3`
- `002_er4.mp3`
- `003_san1.mp3`

命名规则：

- 前三位是汉字编号
- 后面是数字声调拼音
- `1`、`2`、`3`、`4` 分别表示一声到四声
- `5` 表示轻声
- `ü` 使用 `v`，例如 `lǜ` 对应 `lv4`

生成或更新清单：

- `npm run audio:manifest`

检查缺失音频：

- `npm run audio:check`

在 300 个 mp3 补齐之前，检查命令会列出缺失文件并以失败状态退出，这是预期行为。

游戏与 A4 点读页播放逻辑：

1. 如果对应 mp3 存在，优先播放本地音频。
2. 如果音频不存在或播放失败，自动退回浏览器中文朗读。

多音字请按清单里的 `pinyin` 字段录制当前指定读音。
